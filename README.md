# Physics Sandbox PWA

Ball drop + peg board + ragdoll physics sandbox built with React and Vite.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## PWA Notes

- Uses `vite-plugin-pwa` with auto-updating service worker.
- Manifest and Workbox config live in `vite.config.js`.
- PWA icons are in `public/pwa-192x192.png`, `public/pwa-512x512.png`, and `public/apple-touch-icon.png`.
- Install prompt availability depends on browser/platform support.
